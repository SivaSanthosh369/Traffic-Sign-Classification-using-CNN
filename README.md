This project develops a robust image classification system designed to identify 43 distinct categories of German traffic signs, ranging from speed limits to cautionary warnings. Built on the German Traffic Sign Recognition Benchmark (GTSRB) dataset, the solution leverages the power of Convolutional Neural Networks (CNNs) to process visual data efficiently.

The workflow begins with an automated data ingestion pipeline using kagglehub, followed by rigorous preprocessing steps including image resizing to 64x64 pixels and pixel normalization to enhance training stability. The core of the project is a custom-built CNN architecture designed for high-performance feature extraction. To ensure model reliability, the training process incorporates advanced techniques such as L2 Regularization and Dropout to mitigate overfitting, alongside Early Stopping to optimize training duration.

The final model serves as a foundational component for autonomous vehicle systems, providing a scalable and accurate method for environmental perception. By converting raw pixel data into actionable class predictions, this project demonstrates the practical application of deep learning in modern computer vision challenges.
